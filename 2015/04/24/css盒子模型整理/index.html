css盒子模型整理

1 什么是盒子模型？

  • 在网页布局中，为了能够使纷繁复杂的各个部分合理地进行组织，通过研究，总结出了一套完整的、行之有效的原则和规范，这就是“盒子模型”的由来。

  • 所有页面中的元素都可以看成是一个盒子，占据着一定的页面空间。一个独立的盒子所占空间组成部分有内容content，内边距padding，边框border 和外边距margin 组成 。

2 标准盒子模型与怪异盒子模型（ie8以下）

  • w3c标准规定的时标准盒子模型，此时的盒子的宽高为:

    宽/高 = width/height + 左右/上下padding + 左右/上下border-width + 左右/上下margin

  • 由于IE中则采用Microsoft自己的标准，此时的盒子宽高为(怪异盒子模型)：

    宽/高 = width/height + 左右/上下margin （规定width/height = 内容宽/高 + 左右/上下padding + 左右/上下border-width）

3 模型转换

  • 使用c3中的box-sizing 语法：

    box-sizing :content-box(默认标准盒子模型)/border-box(怪异盒子模型)
